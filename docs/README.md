# Docs

This folder captures the v1.0 requirements and design notes for the fork. These documents describe intent and contracts; they are not a statement of completed features.

- `requirements-v1.md` - v1.0 goals, constraints, and scope
- `pipeline-run-and-collect.md` - pipeline tool input/output and step order
- `artifacts.md` - artifact directory layout and manifest fields
- `serial-export.md` - serial command protocol for START/STOP/EXPORT and SD log transfer
- `fork-todo.md` - implementation TODO list in minimum-diff order
- `fork-todo-ja.md` - フォーク後の修正TODO（日本語版）
